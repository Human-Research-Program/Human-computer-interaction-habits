# 人类计算机交互习惯

> 本文档旨在为后续软件人机交互设计提供参考

## 通用(无争议,大部分软件通用)

+ Ctrl + C => 复制
+ Ctrl + V => 粘贴
+ Ctrl + Z => 撤销

## 鼠标

### 原型设计软件

#### Adobe XD

+ 左键单击 => 选中
+ 左键长按并拖动 => 范围选中
+ 中键(滚轮) 按住并拖动 => 移动视图
+ 右键 => 操作菜单

### 3D 设计软件

#### Blender

+ 单击左键 => 选中
+ 滚轮上下滑动 => 放大/缩小视图
